리디렉션: 쉘에서 표준 입출력 스트림을 다른 방향으로 바꾸어주는 기능 <br>

## 출력 리디렉션 >
출력을 다른 곳으로 보내기
```
[n(저장할 스트림)] > [|]filename

ex) ls > filename
```

스트림을 대상 파일에 저장 <br>
- 1: 표준출력
- 2: 표준에러
- 생략 시 표준 출력

<img src="img/img.png" width=700 height=450>

대상 파일이 존재하는 경우 덮어쓰기 <br>
noclobber 옵션 설정 시 덮어쓰기 시도 시 에러 발생

```
> | 
//사용 시 noclobber 옵션 여부와 상관없이 덮어쓰기 성공
```
